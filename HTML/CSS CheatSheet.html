Common CSS Proporties Reference

Property Names & Examples

Name Description Example
Color The color property allows us to specify the text color of an HTML element. This applies to any HTML element that has text content including h1-h6, p, a and more. There are multiple valid formats for color values, however the most common are hex values, rgba and named colors. /* Hex Value */ color: #000000; /* RGBA */ color: rgba(0, 0, 0, 0.3); /* Named Colors */ color: black;
Font-Size The font-size property allows us to change the size of the font for any text based content. a { font-size: 48px; }
Background Color The background-color property allows you to set the background color of an HTML element. Just like text color, the value of this property can be a hex value, rgba or named color. h1 { background-color: #FFB300; }
Height/Width The height and width properties specify the height and width of a HTML element respectively. There are many different values you can apply to both of these properties, but for this tutorial we'll focus on pixels (px) and percentages (%) p { background-color: blue; height: 250px; width: 500px; }
Border The border property allows you set a border for your HTML element. img { border: 5px solid #FF6DBA; }
Background Image The background-image property allows us to add a background image to an HTML element. This is similar to the background color, however an image is shown instead of a color. h1 { height: 300px; color: white; border: 1px solid black; background-image: url("../img/rainbow_bg.jpg"); }
